Calculadora Objetiva


Trata-se de meu primeiro app para iOS completo, que foi a base para meu trabalho de conclusão de curso, obtendo nota máxima. É uma calculadora simples e moderna, que realiza cálculos em tempo real, enquanto o usuário digita os números. É um projeto de código aberto, criado através do padrão de design MVC, utilizando-se do construtor de interface (Interface Builder). Também é uma aplicação universal, suporta múltiplas orientações de tela e até mesmo multi-tarefas para iPad.

InfiniteScrollingBackground


Uma classe feita em Swift para games de iOS e Mac que usam SpriteKit. Ela cria um fundo móvel infinito, excelente para qualquer jogo 2D! É possível adicionar uma quantidade ilimitada de imagens, personalizar a direção de rolagem e até criar diversos objetos com diferentes ZPositions e usá-los simultaneamente.

GameAudioPlayer


Classe que resolve todos os problemas de áudio encontrados por qualquer desenvolvedor que use SpriteKit (Memory Leaks com o método SKAction.PlayFileNamed, por exemplo). Nela, SKAudioNodes são criados dinânicamente, e efeitos sonoros podem ser carregados antes serem utilizados, otimizando a performance.
